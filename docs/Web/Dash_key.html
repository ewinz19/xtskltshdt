Dash.js ClearKey DRM Tester

Alat untuk menguji stream DASH yang dilindungi dengan ClearKey DRM. Masukkan URL MPD dan kunci DRM Anda, lalu klik "Load Stream".

Informasi Penting

Pastikan stream Anda menggunakan enkripsi ClearKey DRM. Contoh ini hanya berfungsi untuk ClearKey, bukan Widevine, PlayReady, atau FairPlay.

Gunakan ini jika Anda memiliki multiple keys. Jika diisi, input Key ID/Value di atas akan diabaikan.
