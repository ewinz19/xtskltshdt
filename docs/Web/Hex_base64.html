Konverter Hex ke Base64 (Dua Output)

Alat ini mengonversi dua data hex yang dipisahkan dengan titik dua (:) ke format base64 terpisah. Masukkan data hex dalam format "hex1:hex2" dan hasil konversi akan muncul sebagai dua output base64 terpisah.

Terjadi kesalahan. Pastikan format input benar (hex1:hex2).

Contoh Format Input:

Format: hex1:hex2

Contoh 1: 69646b755f3130303030303030307930:e4a2359b05563399f1d9afce6488724a

Contoh 2: 0123456789abcdef:fedcba9876543210

Hasil: Dua output base64 terpisah akan dihasilkan untuk masing-masing hex.
